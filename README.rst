============
ht-fetch-ids
============

This repo contains a set of Python 3.9 scripts for taking a CSV of the flavor exported by Innovative Sierra's Create List feature containing bibliographic identifiers like OCLC number, Library of Congress Catalog Numbers, ISBNs and ISSNs, reconciling them using the HathiTrust Bibliographic API and printing a CSV containing HathiTrust Volume IDs for use in creating HathiTrust Research Center Worksets.
